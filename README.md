# Writing Reboot Review（写作复盘）

通过结构化深度对话，识别断更真实原因，澄清写作收益，提炼独特价值，产出可执行的写作定位和重启建议。

## 使用场景

- 写作断更，想重新开始
- 写了很久没起色，想做内容定位
- 不确定写什么、写给谁
- 写作遇到瓶颈，坚持不下去

> 注意：只是要标题、选题、润色，不要跑完整复盘。

## 工作流程

| Step | 主题 | 核心问题 |
|------|------|----------|
| 0 | 时间范围 | 复盘过去多久？ |
| 1 | 成就 | 过去一年最引以为傲的三件事？ |
| 2 | 遗憾 | 可以重来，最想换哪种方式？ |
| 3 | 中断原因 | 真正的原因是什么？ |
| 4 | 对比 | 其它事能坚持，写作为什么不行？ |
| 5 | 收益 | 写作至少要带来什么才愿意继续？ |
| 6 | 定位 | 别人不容易有的经历/视角/方法？ |

## 特殊处理

- 说"不知道" → 问别人最认可你什么、最常问你什么
- 过度否定自己 → 指出事实矛盾，收回"设计不对"还是"目标不对"
- 只想快速结论 → 覆盖四要素：成就+遗憾+中断原因+目标

## 输出结构

1. **用户画像** - 核心优势(≤3) + 真实动机 + 关键阻碍
2. **写作定位** - 一句话标签 + 目标读者 + 内容方向(≤3) + 核心价值
3. **重启建议** - 起步方式 + 节奏 + 成功标准 + 避坑
4. **下一步** - 3个内可执行动作

## 原则

- 一次只问一个问题，不做问卷
- 先回应提炼，再追问
- 不接受"懒""没时间"表层归因
- 不虚构经历和定位
- 只给能立刻执行的建议
